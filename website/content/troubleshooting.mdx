# Troubleshooting

## Installation

| Problem                                                            | Likely cause                                                                                                                                            |
| ------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `command not found` after install script or Homebrew               | `/usr/local/bin` (or your `INSTALL_DIR`) is not on your `PATH`                                                                                          |
| `command not found` after `go install`                             | Add `$(go env GOPATH)/bin` to your `PATH`                                                                                                               |
| Install script fails to download                                   | No release for your OS/arch, or GitHub API rate limit — try [Homebrew](/install#homebrew-macos--linux) or a [manual download](/install#github-releases) |
| `brew install sabinadams/natsmith/natsmith` — repository not found | The tap repo was missing; it lives at [homebrew-natsmith](https://github.com/sabinadams/homebrew-natsmith). Re-run the install command                  |
| `go install …@latest` fails on old tag                             | Use `@v1.2.0` or newer; see [Install](/install#with-go)                                                                                                 |

## Connection

| Problem                             | Likely cause                                                                |
| ----------------------------------- | --------------------------------------------------------------------------- |
| `connect to …` / connection timeout | Wrong URL, network/firewall, or credentials                                 |
| Authentication errors               | Missing or wrong creds in the NATS context — run `nats context info <name>` |
| `unknown context`                   | No matching file in `~/.config/nats/context/` — run `nats context ls`       |

## migrate kv

| Problem | Likely cause |
| ------- | ------------ |
| `destination bucket not found` | Create the KV bucket on destination first (e.g. Terraform) |
| Run appears slow on large buckets | Expected for API copy — use `--workers 32` and `--no-verify`; or use [`backup kv`](/commands/backup/kv) + [`restore kv`](/commands/restore/kv) for bulk stream snapshots |
| Migratable count ≠ `nats kv ls \| wc -l` | Investigate before migrating — counts should match |
| Verify reports missing / mismatch | Re-run `natsmith migrate kv --verify-only --failures-file kv-failures.log …` |
| Verify reports dest-only | Extra keys on destination not present on source |
| `skipped dest-only scan` | Bucket has &gt; 1M keys — inline verify still ran; full dest-only scan was skipped |

See [`migrate kv`](/commands/migrate/kv) for flags and examples.

## migrate objects

| Problem                            | Likely cause                                                                           |
| ---------------------------------- | -------------------------------------------------------------------------------------- |
| `destination bucket not found`     | Create the object store bucket on destination first                                    |
| Scan appears stuck                 | Large meta stream — progress updates every 250 messages; wait or filter with `-bucket` |
| Object store reports omitted       | Meta on source but object data not retrievable (deleted / tombstone)                   |
| Copy failures / `context canceled` | Increase `-timeout` (e.g. `5m`) and/or lower `-workers`                                |
| Link migration fails               | Linked bucket or object not migrated yet                                               |

See [`migrate objects`](/commands/migrate/objects) for flags and examples.
