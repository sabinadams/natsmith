# Install

natsmith is a **global CLI** — install once and run from any directory. It is not a project dependency.

Pre-built binaries are available for macOS, Linux, and Windows. **Go is not required** unless you use [`go install`](#with-go) or [`go run`](#run-without-installing) below.

## How versions work

| Install method | Which version you get |
| -------------- | --------------------- |
| [Install script](#install-script-macos--linux) | Always the latest [GitHub release](https://github.com/sabinadams/natsmith/releases/latest) (resolved when you run the script) |
| [Homebrew](#homebrew-macos--linux) | Latest formula version after `brew update` |
| `go install …@latest` | Latest tagged release via the Go module proxy |
| Pinned examples below | The **current release** shown above — updated when this site is rebuilt after a new release |

You do not need to hunt for version numbers for most installs. Use the install script, Homebrew, or `@latest`. Pin a specific tag only if you need a fixed version.

## Install script (macOS & Linux)

Downloads the latest release binary from GitHub:

```bash
curl -fsSL https://sabinadams.github.io/natsmith/install.sh | sh
```

Install to a different directory:

```bash
INSTALL_DIR="$HOME/.local/bin" curl -fsSL https://sabinadams.github.io/natsmith/install.sh | sh
```

Ensure that directory is on your `PATH`.

## Homebrew (macOS & Linux)

```bash
brew install sabinadams/natsmith/natsmith
```

The formula is updated automatically on each release.

## GitHub Releases

Download a platform archive from [GitHub Releases](https://github.com/sabinadams/natsmith/releases). Each archive contains the `natsmith` binary. Verify downloads with `checksums.txt`.

## With Go

For Go developers who already have Go 1.25+ installed:

```bash
export PATH="$(go env GOPATH)/bin:$PATH"   # add to ~/.zshrc or ~/.bashrc to persist
```

```bash
go install github.com/sabinadams/natsmith/cmd/natsmith@latest
natsmith migrate kv -h
```

## Run without installing

Requires Go 1.25+. Use `go run` and pass the subcommand after `--`:

```bash
go run github.com/sabinadams/natsmith/cmd/natsmith@latest -- migrate kv -dry-run \
  --source-context sandbox-legacy \
  --dest-context sandbox
```

## Updating

Re-run the [install script](#install-script-macos--linux) or:

```bash
brew upgrade natsmith
```

Go users:

```bash
go install github.com/sabinadams/natsmith/cmd/natsmith@latest
```

```bash
go version -m "$(command -v natsmith)"
```

For manual GitHub Release installs, download the newer archive and replace the binary on your `PATH`.
