# backup kv

```bash
natsmith backup kv [flags]
```

Snapshot KV buckets to disk (`backup.json` + `stream.tar.s2`). Same format as `nats stream backup`.

Requires **stream admin** on the source cluster.

## Output

```text
{--dir}/
  <bucket>/backup.json
  <bucket>/stream.tar.s2
```

Omit `--bucket` to backup all KV buckets on the cluster.

## Flags

Inherits all [backup](/commands/backup) flags.

## Examples

### Backup one bucket

```bash
natsmith backup kv \
  --context sandbox-legacy \
  --dir ./backups/sandbox-legacy \
  --bucket schema
```

### Backup all buckets

```bash
natsmith backup kv \
  --context sandbox-legacy \
  --dir ./backups/sandbox-legacy
```
