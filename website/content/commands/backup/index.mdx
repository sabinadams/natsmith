# backup

```bash
natsmith backup <subcommand> [flags]
```

Export JetStream streams to local snapshot files.

## Subcommands

| Command | Description |
|---------|-------------|
| [`kv`](/commands/backup/kv) | Snapshot `KV_<bucket>` streams |

## Global flags

Persistent on `backup`:

| Flag | Default | Description |
|------|---------|-------------|
| `--context` | — | NATS CLI context (required) |
| `--dir` | — | Output directory (required) |
| `--bucket` | all | Comma-separated bucket filter |
| `--omit` | — | Comma-separated buckets to skip |
| `--no-progress` | `false` | Plain stderr |
| `--timeout` | `30s` | Per-request NATS timeout |

Pair with [`restore`](/commands/restore) on the destination cluster. See [KV migration](/guides/kv-migration) for when to use this vs [`migrate kv`](/commands/migrate/kv).

## Example

```bash
natsmith backup kv \
  --context sandbox-legacy \
  --dir ./backups/sandbox-legacy
```
