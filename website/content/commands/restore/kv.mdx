# restore kv

```bash
natsmith restore kv [flags]
```

Restore KV buckets from snapshots created by [`backup kv`](/commands/backup/kv) or `nats stream backup`.

## Input

`--dir` is either:

- A **bucket directory** containing `backup.json` and `stream.tar.s2`
- A **parent directory** with one subdirectory per bucket (from `backup kv` without `--bucket`)

Restores every matching subdirectory in one run.

## Flags

Inherits all [restore](/commands/restore) flags.

## Examples

### Restore all buckets

```bash
natsmith restore kv \
  --context sandbox-dedicated \
  --dir ./backups/sandbox-legacy \
  --force
```

### Restore one bucket

```bash
natsmith restore kv \
  --context sandbox-dedicated \
  --dir ./backups/sandbox-legacy/schema \
  --force
```

## Notes

- Restores **full stream state** (history, tombstones), not live keys only.
- Use `--force` if destination already has the bucket (including after a partial `migrate kv`).
- Creates the backing stream; destination buckets need not exist beforehand.
- Set `--replicas` when destination topology differs from the backup.
