# restore

```bash
natsmith restore <subcommand> [flags]
```

Import JetStream stream snapshots from local files.

## Subcommands

| Command | Description |
|---------|-------------|
| [`kv`](/commands/restore/kv) | Restore `KV_<bucket>` streams |

## Global flags

Persistent on `restore`:

| Flag | Default | Description |
|------|---------|-------------|
| `--context` | — | NATS CLI context (required) |
| `--dir` | — | Backup directory (required) |
| `--bucket` | all | Comma-separated bucket filter |
| `--omit` | — | Comma-separated buckets to skip |
| `--force` | `false` | Delete existing streams before restore |
| `--replicas` | `0` | Override replica count (`0` = use backup) |
| `--no-progress` | `false` | Plain stderr (no progress bars) |
| `--timeout` | `30s` | Per-request NATS timeout |

Root flags [`--quiet`](/commands#global-flags) and [`--json`](/commands#global-flags) also apply.

See [KV migration](/guides/kv-migration) for workflow and when to use `--force`.

## Examples

### Restore all KV buckets

```bash
natsmith restore kv \
  --context sandbox-dedicated \
  --dir ./backups/sandbox-legacy \
  --force
```
