# natsmith

CLI tooling for [NATS](https://nats.io) and JetStream.

Unofficial; not affiliated with Synadia. Not a replacement for the official [`nats` CLI](https://github.com/nats-io/natscli).

## Getting started

1. [Install](/install) natsmith
2. Run `natsmith -h` to see top-level commands
3. Open the [Commands](/commands) reference for flags and examples

```bash
natsmith -h
natsmith migrate -h
```

## Command groups

| Group | Description |
|-------|-------------|
| [`migrate`](/commands/migrate) | Copy JetStream KV buckets and object stores between clusters |

New command groups will appear here as natsmith grows.
