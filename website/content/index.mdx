# natsmith

CLI tooling for [NATS](https://nats.io) and JetStream.

Unofficial; not affiliated with Synadia. Not a replacement for the official [`nats` CLI](https://github.com/nats-io/natscli).

## Getting started

1. [Install](/install) natsmith
2. Run `natsmith -h` to see top-level commands
3. Open [Commands](/commands) for flags, output modes, and examples
4. See [Guides](/guides) for migration workflows

```bash
natsmith -h
natsmith migrate -h
```

## Command groups

| Group | Description |
|-------|-------------|
| [`migrate`](/commands/migrate) | Copy KV buckets and object stores between clusters |
| [`backup`](/commands/backup) | Export JetStream streams to local snapshots |
| [`restore`](/commands/restore) | Import streams from local snapshots |
