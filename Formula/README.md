The Homebrew formula is maintained in the [`homebrew-natsmith`](https://github.com/sabinadams/homebrew-natsmith) tap and updated automatically by GoReleaser when a `v*` tag is published. No manual formula edits are required. See [Releasing](https://github.com/sabinadams/natsmith/blob/main/CONTRIBUTING.md#releasing) in CONTRIBUTING.md.

```bash
brew install sabinadams/natsmith/natsmith
```
